Curso de Bootstrap 4 - Simples página da web
Realizei a criação de uma página de site simples, é uma simples página, mas comporta as mais diversas tecnologias atuais que o Bootstrap v4 pode oferecer, utilizando o NPM e Sass.

Contém:
    • Navbar interativa, apresentando um menu para todo o site, como também para as redes sociais
    • Carrosel ou Slides com imagens e textos que podem ser avançados ou retrocedidos
    • ScrollSpy com índice interativa
    • Cards com link para modals
    • Vídeos embedados em NavPills contidos em um Jumbotron
    • Formulário para inserção de dados
    • List Group com informação do site
    • Button Group como menu de rodapé para o site
    • Blockquote para as redes sociais disponíveis
